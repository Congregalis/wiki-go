# Wiki GO

简单入门 go web ！

配合以下参考文章很快就能做出本项目。
> 参考：http://shouce.jb51.net/golang-doc/53.html

## 功能
- `/view` : 查看某 wiki
- `/edit` : 编辑某 wiki

## 部署

1. 本地普通部署

执行 

`go run wiki.go`

或执行 

`go build wiki.go`

`./wiki`

访问 `http://localhost:8080/` 即可开始
